# CET-6 题库学习网页（纯静态）

## 运行方式

### 方式 1：直接双击打开（推荐给 DW）
直接打开 `index.html` 即可运行（不需要服务器）。

### 方式 2：本地静态服务器（可选）
如果你想更像“网站”那样预览，也可以用任意静态服务器打开本目录。

## 题库导入/导出

1. 进入【题库】页
2. 点击“导出当前题库”得到 JSON
3. 你可以用编辑器/Excel 处理后再导入（导入时会校验结构）

### 每题“小提示”必填
导入题库时会强校验 `hint` 字段：
- 不能为空
- 建议写“线索/定位/方法”，不要直接写“答案是X”

## 本地数据说明
学习记录、错题本、收藏等数据保存在浏览器 `localStorage`（只在本机有效）。
如需清空，可在【题库】页点击“重置学习数据”，或清除浏览器站点数据。
